Use THIS client for translation perf testing. It targets the MIAW_Deployment / MIAW_Messaging_Channel deployment, which routes to a human agent via Omni-Channel (the Translate world-icon lives on the human agent console). The other client (miawTranslate_*.html, deployment MIAWASADep / MIAWWithASA) routes to the Agentforce Service Agent bot instead, so a session there never reaches the Omni inbox — wrong target for this work.

Click the chat widget (bottom-right) and send an inbound message in the selected non-English language (e.g. Spanish "Hola, necesito ayuda con mi pedido") to trigger language detection and the agent-side Translate world-icon. The picker sets the END-USER (customer) locale; the agent works in English. Choose en_US as a control case where the icon should NOT appear.
