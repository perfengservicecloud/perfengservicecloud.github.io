Release-262 org (Org Id 00DVW000009TofO, cell sdb39) — the more-stable replacement for the 264 org. Use THIS client for translation perf testing: it targets the MIAWWithASA deployment, whose channel routes sessionHandlerType=Queue straight to MessagingQueue — a human agent via Omni-Channel (the Translate world-icon lives on the human agent console). NAMING IS INVERTED vs the 264 org: on 262 the MIAWDeployment / MIAWChannel deployment routes through flow Route_to_Astro_Enhanced to the Astro Enhanced bot (session owner "Automated Process", never reaches the Omni inbox) — so despite the "ASA" name, MIAWWithASA is the human one here.

Click the chat widget (bottom-right) and send an inbound message in the selected non-English language (e.g. Spanish "Hola, necesito ayuda con mi pedido", or Chinese "你好，我需要帮助") to trigger language detection and the agent-side Translate world-icon. The picker sets the END-USER (customer) locale; the agent works in English. Choose en_US as a control case where the icon should NOT appear.
