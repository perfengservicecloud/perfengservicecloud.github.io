Release-262 org (Org Id 00DVW000009TofO, cell sdb39) — the more-stable replacement for the 264 org. Use THIS client for translation perf testing: it targets the MIAWDeployment / MIAWChannel deployment, which routes to a human agent via Omni-Channel (the Translate world-icon lives on the human agent console). The other 262 deployment (MIAWWithASA) routes to the Agentforce Service Agent bot instead, so a session there never reaches the Omni inbox — wrong target for this work.

Click the chat widget (bottom-right) and send an inbound message in the selected non-English language (e.g. Spanish "Hola, necesito ayuda con mi pedido") to trigger language detection and the agent-side Translate world-icon. The picker sets the END-USER (customer) locale; the agent works in English. Choose en_US as a control case where the icon should NOT appear.
