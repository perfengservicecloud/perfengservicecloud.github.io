Client for the GCP/AWS translation UI perf work. Click the chat widget (bottom-right) and send an inbound message in the selected non-English language (e.g. Spanish "Hola, necesito ayuda con mi pedido") to trigger language detection and the agent-side Translate world-icon. The picker sets the END-USER (customer) locale; the agent works in English. Choose en_US as a control case where the icon should NOT appear.
